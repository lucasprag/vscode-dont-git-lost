# Git Lost

Step through any file's history in the editor. Subdued inline blame with rich hover. Lightweight.

## Features

- File time-travel: ◀ ▶ buttons in the editor toolbar step through every commit that touched the current file.
- Inline blame: the line under the cursor is annotated with author, message, and relative time.
- Hover card: rich commit details with author avatar, commit link, PR link, and copy-SHA button.
